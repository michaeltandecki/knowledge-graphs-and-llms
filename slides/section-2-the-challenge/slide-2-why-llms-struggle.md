## Why LLMs Struggle

Tokens are not concepts. Context windows are scratchpads, not long-term memory.

- Text ≠ entity. Names ≠ IDs.
- Models store statistical patterns implicitly; you cannot query or audit those "facts" reliably
- Long context helps only inside the current session
- Consistency across sessions requires an external memory system

Note:
- Analogy: context window = scratchpad; graph = persistent memory.
- "Semantics without identification is philosophy without physics." — Tony Seale (user-provided excerpt).
